又是一学期

转眼一学期又过去了。

原本不抱希望的课程得了A，很惊喜。

不管怎么样都过去了。

今天是第一次跟某些人出去玩，估计也是最后一次了。

用方言说，我现在是“yanzhente”太多。

想了很久都不知道是哪三个字的缩写。

10月 3 2014

我们都是小孩子

最后一天一个人在寝室的日子。果然还是最喜欢一个人的生活。

好像一下子开朗了很多，连早晨的闹钟也变得特别温暖。

我们都是小孩子，过去所有的艰难不开心都是为了以后能够和另一个人过更好的生活。

好吧，很多话都不想写了，好懒好懒。
